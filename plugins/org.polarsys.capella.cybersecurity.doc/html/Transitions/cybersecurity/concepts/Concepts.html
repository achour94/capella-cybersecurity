Concepts

The Cybersecurity Viewpoint allows the top-down transition of cybersecurity elements (Threats and Primary Assets) and their dependencies. For more details on transitions, check out the Capella Documentation on the subject.

Performing a transition

Transition of cybersecurity elements can be done using the "Cybersecurity Transition" command available in the transitions menu.

The cybersecurity transition command is available on the following elements:

  • On the architecture levels (Operational, System or Logical)
  • On the cybersecurity package
  • On the cybersecurity elements (Threats and Primary Assets)
