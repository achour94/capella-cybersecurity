Concepts

The DARC Viewpoint allows the top-down transition of cybersecurity elements (Threats and Primary Assets) and their dependencies. For more details on transitions, check out the Capella Documentation on the subject.

Performing a transition

From Activity Explorer

The user can perform automated transitions of Cybersecurity Threats and Assets for SA, LA and PA levels using the Perform an automated transition of Cybersecurity elements tool, available in Activity Explorer. The tool will transition the content of Cybersecurity package (which contains Threats and Primary Assets).

Using the transitions menu

Transition of cybersecurity elements can be done using the Cybersecurity Transition command available in the transitions menu (in Project Explorer or in a diagram):

The cybersecurity transition command is available on the following elements:

  • On the architecture levels (Operational, System or Logical)
  • On the cybersecurity package
  • On the cybersecurity elements (Threats and Primary Assets)
